# ThoughtHarbour

## Project Status: Work in Progress

This project is currently under development and various functionalities are being explored and added incrementally. It serves primarily as a learning venture and a practical exercise in using Blazor and MudBlazor for web application development. Feedback and suggestions are most welcome!

## Deployment Status: Not Hosted

Please note that as of now, ThoughtHarbour is not hosted on any platform, and hence, is not accessible for online use or viewing. The database connectivity is also set up for local development and won’t be accessible for any forked/cloned versions of this repository without proper setup.

## Overview

ThoughtHarbour is a simplified task management application aimed at facilitating a hands-on learning experience in working with Blazor and MudBlazor. The primary objective is to explore various functionalities and implementations possible with the aforementioned technologies.

## Setup and Usage

After cloning the repository:

- Ensure that you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
- Navigate to the project directory and run `dotnet run` or use an IDE like Visual Studio to run the project.
- Configure a local database and update the connection string in the `appsettings.json` file to get started with exploring the app on your local machine.

## Contribution

While ThoughtHarbour is a personal learning project, any contributions, suggestions, and feedback are highly welcomed! Feel free to fork the project and explore your own implementations.

## Technologies Used

- **Blazor:** For handling the client-side functionality and creating interactive web UIs.
- **MudBlazor:** A popular Material Design component library providing a rich set of UI components.
